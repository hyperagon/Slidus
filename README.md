## Controls
- **Space** / **Click circle** (behind toolbar) : Change Mode
- **x** / **Click on Circle** : Show Toolbar

### Any Mode
- Undo: **z** / **Ctrl+z**
- Reso: **y** / **Ctrl+y**
- Copy: **Ctrl+c**
- Paste: **Ctrl+v**
- Duplicate: **d** / **Ctrl+d**
- Move selection: Drag **Cross**
- Rotare path: Drag **Circled X**, if holding Ctrl you'll drag the pivot
- Zoom In: **+** / **i** / **spread fingers**
- Zoom Out: **-** / **o** / **pinch fingers**
- Pan: **hold right mouse-button + drag** / **2 finger drag**
- The browsers own zoom still works with *Ctrl+Mouse Wheel* which is misleading as it is not handled by the Application as zooming

### Select Mode
- White Square = **Anchor**, drag these by pressing and moving while holding
- Red Circle = **Control Node**, these adapt if you drag the path
- Dragging **Anchors** takes its **Control Nodes** along
- Hold Ctrl or Cmd to snap **Control Handles** to multiples of 5
- Double-click or Ctrl+Click **Anchor** to delete it
- Double-click path to add **Anchor**
- Drag **Anchor** near paths or **Anchors** to snap
- Select a path and either click "**Close / Merge**" or drag its start to its end (Closing adds *Control Handles**)
- Select multiple paths and click "**Close / Merge**" to connect and mergre into one
- **Copy** a color from the reference by pressing the button between the colors
 then dragging over/clicking a color box and clicking on the reference
- **Trace** a color by pressing the "Yrace" button then the color to trace on the reference image
