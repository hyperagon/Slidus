## Controls
- **Space** / **Click circle** (behind toolbar) : Change Mode
- **x** / **Click on Circle** : Show Toolbar

### Any Mode
- **z** / **Ctrl+z** : Undo
- **y** / **Ctrl+y** : Redo
- **+** / **i** / **pinch fingers** : Zoom In
- **-** / **o** / **spread fingers** : Zoom Out
- **hold right mouse-button + drag** / **2 finger drag** : Pan

### Select Mode
- White Square = **Anchor**,dragging these takes its **Control Nodes** along
- Red Circle = **Control Node**, these are dragged if you drag the path
- 

<b>Red Circles</b> = <span style='color:#e74c3c'>Control Handles</span>.<br>
                Drag <span style='color:#e74c3c'>Control Handles</span> to curve paths.</br>
                Hold Ctrl or Cmd to snap <span style='color:#e74c3c'>Control Handles</span> to multiples of 20.</br>
                <b>White Squares</b> = <span style='color:var(--accent-color)'>Anchor</span>.<br>
                Double-click <span style='color:var(--accent-color)'>Anchor</span> to delete it.<br>
                Double-click path to add <span style='color:var(--accent-color)'>Anchor</span>.<br>
                <b>Snapping:</b> Drag <span style='color:var(--accent-color)'>Anchor</span> near paths or <span style='color:var(--accent-color)'>Anchors</span> to snap.<br>
                <b>Close:</b> Select a path and either click "<b>Close / Merge</b>" or drag its start to its end. (Closing adds <span style='color:#e74c3c'>Control Handles</span>)</br>
                <b>Merge:</b> Select multiple paths and click "<b>Close / Merge</b>".
